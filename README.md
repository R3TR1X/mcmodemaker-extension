# MC Mod Maker

A specialized VS Code extension for creating and scaffolding perfect Minecraft Mod environments directly within the editor!

## Features

- **Fabric Scaffolding**: Dynamically generates mods using real-time mappings matching the latest Fabric configurations for any Minecraft version.
- **Java and Kotlin Setup**: Automatically generate your mod entrypoints perfectly suited to your language of choice.
- **Mod Tools**: 
  - Effortlessly add new **Commands** and execute them directly with smart Kotlin & Java matching.
  - Listen in on major events with the **Event Listener Generator**.
  - Configure options for your community using the **Config File Generator**, using advanced GSON tracking.

## Usage

1. Click on the Creeper Icon in the Activity Bar.
2. Select **Create Mod** from the sidebar to start a new project!
3. Right click the tools to scaffold your environment!

Enjoy making your Minecraft Mods natively in VS Code/VSCodium!
